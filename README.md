# Minimalist Magazine Website

A clean, minimalist website for a magazine with integrated shopping functionality. This project features a modern design with Courier New as the main typeface, emphasizing generous white space and smooth user experience.

## Features

- **Minimalist Design**: Clean layout with a focus on content and typography
- **Responsive**: Fully responsive design that works on all devices
- **Online Shop**: Integrated e-commerce functionality for selling magazine issues and merchandise
- **Article Browsing**: Categorized articles with filtering capability
- **Contact Form**: Interactive form for visitor inquiries
- **Smooth Animations**: Subtle animations enhance the browsing experience

## Pages

1. **Home**: Introduction to the magazine with featured articles and products
2. **Articles**: Browse and filter articles by category
3. **Shop**: View and purchase magazine issues and merchandise
4. **About**: Information about the magazine, team, and contact form

## Tech Stack

- HTML5
- CSS3 (with custom variables for theming)
- Vanilla JavaScript
- Responsive design using CSS Grid and Flexbox

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser to view the website

No build tools or dependencies are required as this is a pure HTML/CSS/JS project.

## Customization

### Styling

The website uses CSS variables for easy customization. The main variables are defined in `styles/main.css`:

```css
:root {
    --primary-font: 'Courier Prime', monospace;
    --text-color: #333;
    --background-color: #fff;
    --accent-color: #f0f0f0;
    --border-color: #e0e0e0;
    --link-color: #555;
    --link-hover-color: #000;
}
```

Modify these variables to change the overall look and feel of the website.

### Adding Content

To add new articles or products:

1. Copy an existing article/product card structure from the HTML
2. Update the content and data attributes
3. Add appropriate categories for filtering

## Design Principles

This website follows minimalist design principles:

- Content-first approach
- Generous white space
- Limited color palette
- Typography as a design element
- Subtle animations
- Intuitive navigation
